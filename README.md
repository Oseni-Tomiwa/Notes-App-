📝 Notes App

A clean, responsive Notes App built with HTML, CSS, and Vanilla JavaScript. It allows users to create, edit, delete, search, and store notes locally in the browser using localStorage.

✨ Features

* 📝 Create new notes
* ✏️ Edit existing notes
* 🗑️ Delete notes
* 🔍 Search notes by title or content
* 💾 Automatically save notes using localStorage
* 🌙 Dark mode with saved preference
* 📱 Fully responsive design
* 📊 Live note counter

🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* localStorage API

🚀 Getting Started

1. Clone the repository:

git clone https://github.com/YOUR_USERNAME/notes-app.git

2. Navigate to the project folder:

cd notes-app

3. Open index.html in your browser.

No additional installation or dependencies are required.
